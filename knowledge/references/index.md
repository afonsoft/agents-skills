## Reference File Guidelines

### Purpose

Reference files contain external documentation formatted for LLM consumption. They should:

- Be concise and focused
- Use clear structure
- Include examples
- Be up-to-date
- Be platform-specific when needed

### Naming Convention

Use descriptive names with `-llms.txt` suffix for LLM-optimized references.

### Maintenance

- Update when external documentation changes
- Validate accuracy monthly
- Remove outdated references
- Add new references as needed
