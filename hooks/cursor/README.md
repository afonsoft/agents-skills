# Cursor Hooks

## Arquivos

| Arquivo | Funcao |
|---------|--------|
| `hooks.json` | Configuracao de hooks do Cursor (sessionStart) |
| `session-start` | Script que injeta catalogo de skills no inicio da sessao |

## Instalacao

```bash
./install.sh --cursor
```

Copia hooks para `~/.cursor/hooks/`.
