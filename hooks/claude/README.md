# Claude Code Hooks

## Arquivos

| Arquivo | Funcao |
|---------|--------|
| `hooks.json` | Configuracao de hooks do Claude Code (SessionStart) |
| `session-start` | Script que injeta catalogo de skills no inicio da sessao |

## Instalacao

```bash
./install.sh --claude
```

Copia hooks para `~/.claude/hooks/`.
