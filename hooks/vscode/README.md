# VS Code / Copilot Hooks

## Arquivos

| Arquivo | Funcao |
|---------|--------|
| `hooks.json` | Configuracao de hooks do Copilot (sessionStart) |
| `session-start` | Script que injeta catalogo de skills no inicio da sessao |

## Instalacao

```bash
./install.sh --vscode
```

Copia hooks para `~/.github/hooks/`.
